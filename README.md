Resalet Farah 💐 – Social Development Platform

Resalet Farah is a modern web platform for a social development foundation, focused on delivering awareness programs, community services, and support initiatives through a clean UI and scalable frontend architecture.

The project emphasizes performance, usability, and maintainability, following modern frontend best practices.

⸻

🚀 Features

🌐 Platform Content
	•	About Us section
	•	Services overview
	•	Gallery for activities, events and services
	•	Directors page
	•	Success Stories showcase
	•	News & Events updates

🤝 User Interaction
	•	Volunteering form
	•	Form validation with user-friendly feedback

🎨 UI / UX
	•	Fully responsive design across all devices
	•	Clean and modern interface
	•	Reusable UI components
	•	Consistent styling system

📦 Data & Integration
	•	API integration
	•	Dynamic data rendering
	•	Scalable component architecture

🔄 Upcoming: Donations system with payment integration

⸻

🌐 Live Demo

https://resalet-farah.vercel.app/

🧱 Tech Stack
	•	Frontend: Next.js (App Router)
	•	Language: TypeScript
	•	Styling: Tailwind CSS
	•	UI Components: shadcn/ui
	•	State Management: Redux Toolkit
	•	Forms: Formik + Yup
	•	Package Manager: Yarn
